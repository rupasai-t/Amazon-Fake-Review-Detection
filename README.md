# Amazon-Fake-Review-Detection
# 🛒 Amazon Fake Review Detection System

## 📌 About
NLP-based Machine Learning model to detect 
fake reviews on Amazon using TF-IDF and 
Logistic Regression.

## 📊 Results
- ✅ Accuracy: 98.7%
- ✅ Dataset: 40,432 Amazon Reviews
- ✅ Fake(CG): 20,216 | Real(OR): 20,216

## 🛠️ Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Google Colab
- Pandas, Matplotlib, Seaborn
